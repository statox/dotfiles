# Markdown Preview Styles

Custom CSS for [iamcco/markdown-preview.nvim](https://github.com/iamcco/markdown-preview.nvim), themed to match the [nightfox](https://github.com/EdenEast/nightfox.nvim) colorscheme.

- `style.css` — page layout and typography (`mkdp_markdown_css`)
- `highlight.css` — syntax highlighting inside code blocks (`mkdp_highlight_css`)

Configured in `lua/plugins.lua` via `vim.g.mkdp_markdown_css` and `vim.g.mkdp_highlight_css`.
